\begin{abstract}
This document describes the system which will be build for S7  advanced telecommunications.
Specifically, it describes a multiplexer/demultiplexer which can be used to transmit more 
than one instance of data at the same time. This can be done by adding wave forms
without any mmanipulation or by first converting it to an electronic signal, then multiplexing, 
and finally sending out the final signal as light again.
\end{abstract}
